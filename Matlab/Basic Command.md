# Array && Matrix

### Row vector
空格分开
```
a = [1 2 3 4 5]
```

### Column Vector
用;分开
```
a = [1; 2; 3; 4]
```

### Matrix
```
a = [1 2;3 4]
```
也有快速的方法
```
a(1:2; 1)
显示 row从第一个到第二个col第一个
1 2
```
---

  - size() : 给出matrix的col和row的数量
  - ' ： transpose矩阵
  - inv(matrix) : 求逆矩阵
