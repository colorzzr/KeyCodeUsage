# 5-过渡动画

`hover` 赋予元素动作特效

### 直接过度

直接调整元素的大小或者是颜色，较为硬核


### 间接过渡

只在CSS3里面支持

  - `transition-delay`:设置盐池
  - `transition-duration`: 设置过渡
  - `transition-property`: 当设置了之后，设置属性会出现渐变，而未设置的不会改变
  - `transition-timing-function`: 贝塞尔曲线, `ease`正常的 `ease-out`由快到慢的 `ease-in`由慢到快 `ease-in-out` 进场出场效果一样快`
