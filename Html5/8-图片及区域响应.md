# 图片

`<img>` -> image

属性：
  - src: 图片源，显示的图片
  - alt: alternative tag， 当图裂了之后显示的文本
  
  
# 区域响应

当想给图片赋予跳转链接的功能时

与`<a>`标签混用，实现超链接跳转

```
<a herf="www.baidu.com">
  <img src="hhhh" />
<a/>
```

## 指定区域响应

用<map>进行引导

属性
  
  - name:类似于id，给img指定使用

子元素
  
  - <area>
  
  属性:
    - herf: 选择url跳转
